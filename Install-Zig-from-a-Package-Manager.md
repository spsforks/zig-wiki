Note: For Linux you can [download static builds](https://ziglang.org/download/).

## Homebrew

```
brew install zig
```

## NixOS

```
nix-env -i zig
```

## Windows (using [scoop](http://scoop.sh/))

```
scoop bucket add extras
scoop install ziglang
```

[Zig AUR](https://aur.archlinux.org/packages/zig/)
